FeathersJS API

Welcome to your FeathersJS API! The server is running successfully.

Available Endpoints:

GET /users

Get all users

GET /users/:id

Get a specific user by ID

POST /users

Create a new user

PUT /users/:id

Replace a user

DELETE /users/:id

Delete a user

POST /dms/OrderCreation/MY/:company

Create a new order for the specified company

POST /api/util/kubeops/submission/v2/uploadSubmissionFileBase64

Upload a file submission with base64 encoding

POST /procu-utils/validate-file-content

Validate file content

POST /procu-utils/concat-strings

Concatenate strings

POST /procu-utils/getDateTime

Get formatted date and time

POST /dms/UserPODept/MY

Get user PO department information
